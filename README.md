# Identify-and-Remove-Suspicious-Browser-Extensions-task-7
Reviewed browser extensions to identify suspicious or unused add-ons, checked permissions and sources, removed unnecessary extensions, restarted the browser, and confirmed normal performance to improve security and safe browsing awareness.

## Objective
Learn to identify potentially harmful browser extensions, review their permissions, and remove any unused or suspicious extensions.

## Tools Used
- Google Chrome / Mozilla Firefox
- Browser Extensions Manager

## Task Description
This task focuses on browser security and safe extension management. The internship guide asks learners to open the browser's extension manager, review installed extensions, check permissions and reviews, identify unused or suspicious extensions, remove unnecessary ones, restart the browser, and document the steps taken.

## Steps Performed
1. Opened the browser extension manager.
2. Reviewed all installed extensions one by one.
3. Checked permissions requested by each extension.
4. Verified whether the extensions were actively used.
5. Looked for suspicious signs such as excessive permissions, poor ratings, or unknown publishers.
6. Removed unused or unnecessary extensions.
7. Restarted the browser.
8. Confirmed that the browser was working normally after cleanup.

## Observations
- Some extensions requested broad permissions such as access to all websites.
- Unused extensions can increase browser risk and reduce performance.
- Extensions should only be installed from trusted sources with good reviews.

## Result
The installed browser extensions were reviewed successfully. Unused and unnecessary extensions were identified and removed where needed. After cleanup and restart, the browser worked normally. This task improved understanding of extension permissions, browser security risks, and safe extension management.

## Outcome
This activity increased awareness of:
- browser security risks
- extension permissions
- malware-related threats
- safe browsing best practices
